I released a sample project to GitHub. Kind of hesitated, because about a few months back, I was a bottom-feeder complacent developer, and my GitHub is testament to that. But **** that, right? I realized GitHub is a great way to track your own progress: how you learn, how much you've learned, answers to your own questions in commit messages, yada yada. So what if my other (never-completed-project) repositories are crap? At least it can be a way to compare how far I've improved. HAHA!

Oh... So anyway, the project is here: bytehala/flow-mortar-gradle-example

It's actually the first repo that I'm willing to share, and it feels great.

This post is not a tutorial on how to use the libraries. I know that a tutorial is much needed, because Mortar requires some tightly coupled classes for it to work and it's hard for beginners to delve into, but I figured out that I learn by doing. I learned how to use Flow and Mortar by piecing things together, diving into the git history, reading code, but most importantly doing actual coding work.

Flow

Flow is Square's own library for managing your application's backstack. I've had to wrestle with FragmentManager's backstack for an app I did and I think I lost. So I tried Flow out, and I think it's magic!

Mortar

I'm not really sure what Mortar is, but from what I understand, it's gives you a simpler lifecycle for your application components. It also simplifies architecting Dagger Modules for you, as well as subgraphs. Mortar also requires for each component to have Presenters in which you'll put business logic, effectively decoupling it from your View code. The Presenters can also be saved into a Bundle. I think.

Why Did I Do This?

I did it because I want need to use Flow in a new project.

I did it because I wanted to learn Mortar.

I did it because Mortar's sample project is not Gradle-ready as of this writing. It's actually outdated and references deprecated and/or deleted classes from the Flow sample project.

I did it because I DID IT. I was able to make the sample project!
