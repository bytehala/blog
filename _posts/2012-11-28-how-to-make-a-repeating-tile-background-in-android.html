I have to make this post because as evidenced by this stackoverflow, people chose to do it the hard way. And yes, I'm calling questions on stackoverflow "stackoverflows".

Step 1: Have your base tile in your res/drawable folder. It should be seamlessly tileable if you want it to look good.

Step 2: Create an xml file in your res/drawable folder. Name it whateveryouwant.xml.

Step 3: Paste this into whateveryouwant.xml:

<bitmap
xmlns:android="http://schemas.android.com/apk/res/android"
android:src="@drawable/base_tile"
android:tileMode="repeat"
android:dither="true" />

Note that android:src's value doesn't include the extension of the base tile. It works as long as your base tile is a supported image file type.
