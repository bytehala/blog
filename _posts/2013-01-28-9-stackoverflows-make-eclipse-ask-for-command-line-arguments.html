Sometimes you need to create a program that reads command line arguments. Eclipse is the IDE of choice for a lot of programmers, and having to switch between cmd and eclipse is just daunting/repetitive/unnecessary/insert-negative-word-here.

To make eclipse prompt you for command line arguments, do this:

[caption id="attachment_70" align="aligncenter" width="346"] Like thissss![/caption]

In SEO friendly version (as if!): Go to Run>Run Configurations>Click the (x)=Arguments tab, type in ${string_prompt}.

There are other options, if you click the "Variables" button, below the Program arguments text field.
